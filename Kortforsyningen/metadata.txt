[general]
name=Kortforsyningen
qgisMinimumVersion=2.0

The purpose of Kortforsyningen plugin is to make Web Map Services (WMS) from Kortforsyningen (Danish Map supply) easily available for use in QGIS Desktop.
In order to use the plugin, a user at Kortforsyningen.dk is required. 
Enter your username and password in the plugin setting dialogue. 
If you do not have a Kortforsyning user please create one using the link in the plugin setting dialogue.
The plugin is developed by the Agency for Data Supply and Efficiency (fka Danish Geodata Agency)
(www.kortforsyningen.dk, kortforsyningen@gmail.com)

version=0.1
author= Danish Agency for Data supply and Efficiency
email= kortforsyningen@gmail.com

about=
This plugin makes it easy to use the WMS-service from Danish Map supply aka Kortforsyningen.
A new menu is added to QGIS which contains WMS' ordered by category.
The plugin is developed by Danish Map supply (Agency for Data Supply and Efficiency).
